a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git es un sistema de control de versiones que nació un poco por casualidad y tiene un origen que resulta curioso.
Linux tenía un gestor de paquetes que pasó a ser de pago, por lo que el desarrollador principal de Linux (Chris Wanstrath) decidió crear un nuevo gestor de control de versiones para su sistema, y que diera soporte a su core. De esta forma nació Git.
